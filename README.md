## Invisible STAR Interview Builder (Government Jobs)

A lightweight CLI that helps you prepare for structured government interviews by turning your raw experience into duty-aligned STAR responses.

### What it does
- Takes a job posting (file or raw text)
- Extracts duty statement percentages
- Maps your experience bullets to each duty
- Generates natural-sounding STAR answers (with anti-robotic phrasing)

### Quick start
```bash
python3 invisible_star_builder.py \
  --job sample_job_posting.txt \
  --experience sample_experience.txt \
  --top 4
```

Optional: write output to a markdown file.

```bash
python3 invisible_star_builder.py \
  --job sample_job_posting.txt \
  --experience sample_experience.txt \
  --top 4 \
  --out interview_pack.md
```
